Home Store 🏠 💰

Este es un proyecto de una tienda de ecommerce para el hogar, llamada "Home Store". El objetivo es ofrecer una amplia variedad de productos para todas las áreas y necesidades del hogar, desde muebles y decoración hasta electrodomésticos y utensilios de cocina.

---

Uso de Formspree.io

Para gestionar los formularios de contacto del sitio, se ha integrado Formspree.io. Este servicio simplifica la recolección de datos de formularios sin necesidad de escribir código backend. Al enviar un formulario, los datos se dirigen a una dirección de correo electrónico específica asociada a la cuenta de Formspree.io, garantizando una comunicación fluida con los usuarios.
